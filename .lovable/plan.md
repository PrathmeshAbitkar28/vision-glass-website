

# Vision Glass Creation — 5-Page Professional Website

## Overview
A premium glassmorphism-themed website for Vision Glass Creation, a Pune-based window and glass solutions company owned by Pratap Bhagwanrao Kathare. Light, fresh color palette with corporate feel, Poppins font, and smooth animations throughout.

---

## Global Elements (All Pages)

### Sticky Navbar
- Left: "VG" logo icon + "Vision Glass Creation" text
- Right: Navigation links (Home, Services, Gallery, About, Contact) + "Contact Us" CTA button
- Mobile: Hamburger menu with slide-in drawer

### Floating Action Buttons (bottom-right)
- WhatsApp button (green) linking to wa.me/919921917083
- Location pin button above it linking to Google Maps address

### Footer (dark background, 4 columns)
- Column 1: Logo, tagline, owner name
- Column 2: Page links
- Column 3: Service links
- Column 4: Phone numbers, email, address, auto-generated QR code via api.qrserver.com
- Copyright bar at bottom

---

## Design & Technical Details

- **Theme:** Glassmorphism + clean corporate, light & fresh colors
- **Font:** Poppins via Google Fonts
- **Icons:** Lucide React only — no emojis anywhere
- **Images:** All from Unsplash as specified in the brief
- **Animations:** Fade-in on scroll, smooth scrolling, hero scroll arrow animation
- **Responsive:** Mobile-first, fully responsive across all breakpoints
- **Routing:** React Router for all 5 pages
- **Content:** All real — no lorem ipsum
