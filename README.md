# Vision Glass Creation

A modern web application for Vision Glass Creation, providing glass and window solutions in Pune.

## Features
- Expert glass partitions & structural facades
- Aluminium & UPVC window systems
- LED mirrors & decorative glass solutions
- Admin dashboard for content management (Projects, Services, Gallery, Inquiries)
- Built with React, TypeScript, Tailwind CSS, and Firebase

## Tech Stack
- **Frontend**: React (Vite), TypeScript, Tailwind CSS, shadcn-ui, GSAP
- **Backend/Database**: Firebase (Firestore)
- **Icons**: Lucide React
- **Management**: TanStack Query (React Query)

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

### Production Build
To create a production build:
```sh
npm run build
```
